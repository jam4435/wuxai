# 功法提取任务

你是一个武侠小说功法提取助手。请阅读以下小说片段，提取所有出现的功法信息。

## 输出格式要求

每个功法按以下JSON格式输出：

```json
{
  "功法名称": "xxx",
  "类型": "内功/外功/轻功/剑法/刀法/拳掌/指法/暗器/枪戟/棍锤",
  "功法品阶": "粗浅/传家/上乘/镇派/绝世/传说",
  "功法描述": "30字以上的详细描述",
  "修炼限制": {
    "属性名": 数值要求（可选）
  },
  "战斗系数": {
    "属性列表": [
      { "属性": "属性名", "系数": 数值, "乘境界": true/false }
    ]
  },
  "特性": {
    "掌握程度": "特性描述"
  }
}
```

## 品阶判断标准与代表功法

### 粗浅（系数总和0.6）

**定义**：庄稼把式、军旅基础，普通人可学，无特殊要求。

| 功法       | 说明                                                 |
| ---------- | ---------------------------------------------------- |
| 吐纳法     | 道家最基本的呼吸节奏，能养生，攒不出什么内力。       |
| 莽牛劲     | 靠吃肉和举石锁练出来的力气，外门硬功的基础。         |
| 太祖长拳   | 宋朝军体拳，招式工整，虽基础却是许多高深武学的根基。 |
| 五虎断门刀 | 走镖、看家护院首选，凶猛但后劲不足，破绽多。         |
| 草上飞     | 依靠借力踩草奔跑，不能真正腾空。                     |

### 传家（系数总和0.8）

| 功法       | 说明                                                 |
| ---------- | ---------------------------------------------------- |

| 抱元劲     | 道家筑基法门，许多二流高手一辈子就卡在这个境界。     |
| 松风剑法   | 青城派剑法，如松之劲，如风之迅，欺负笨重对手很有效。 |

### 上乘（系数总和1.0）

## 修炼限制设计指南

### 可用属性（7维）

角色创建时分配，范围0-20，基本不变动：

| 属性 | 说明                               |
| ---- | ---------------------------------- |
| 臂力 | 力量相关                           |
| 根骨 | 体质相关                           |
| 机敏 | 敏捷相关                           |
| 洞察 | 感知相关                           |
| 悟性 | 学习相关（特殊：影响修为消耗折扣） |
| 风姿 | 魅力相关                           |
| 福缘 | 运气相关                           |

### 属性分档与等级

| 数值范围 | 等级 | 说明                   |
| -------- | ---- | ---------------------- |
| 1-4      | 愚钝 | 该属性极差，有负面影响 |
| 5-8      | 普通 | 常人水平，无特殊加成   |
| 9-12     | 人才 | 略高于常人，无特殊加成 |
| 13-16    | 天才 | 天赋异禀，有正面加成   |
| 17-20    | 妖孽 | 百年难遇，有强力加成   |

### 修炼限制设计原则

1. **粗浅功法**：通常无修炼限制，普通人即可学习
2. **传家功法**：可能有1个属性的基础要求（如臂力≥5）
3. **上乘功法**：通常有1-2个属性要求，门槛在"普通"级别（5-8）
4. **镇派功法**：通常有1-2个属性要求，门槛在"人才"级别（9-12）
5. **绝世功法**：可能有多个属性要求，部分门槛达到"天才"级别（13-16）
6. **传说功法**：可能有极高要求，或需要特殊机缘，门槛可达"妖孽"级别（17-20）

### 修炼限制示例

```json
// 粗浅·太祖长拳 - 无限制
"修炼限制": {}

// 传家·混元一气功 - 基础根骨要求
"修炼限制": {
  "根骨": 5
}

// 上乘·紫霞神功 - 需要一定悟性
"修炼限制": {
  "悟性": 8
}

// 镇派·纯阳无极功 - 需要良好的根骨和悟性
"修炼限制": {
  "根骨": 10,
  "悟性": 9
}

// 绝世·独孤九剑 - 需要极高的洞察力和悟性
"修炼限制": {
  "洞察": 14,
  "悟性": 12
}

// 传说·太玄经 - 需要妖孽级悟性
"修炼限制": {
  "悟性": 17
}
```

### 属性与功法类型对应

| 功法类型    | 常见限制属性 | 说明                                           |
| ----------- | ------------ | ---------------------------------------------- |
| 内功        | 根骨、悟性   | 根骨决定能否承受真气运转，悟性决定能否领悟心法 |
| 外功/拳掌   | 臂力、根骨   | 需要力量基础和强健体魄                         |
| 剑法        | 机敏、洞察   | 需要灵活身手和敏锐观察                         |
| 刀法/重武器 | 臂力、根骨   | 需要足够力量驾驭重兵器                         |
| 轻功        | 机敏         | 需要身体协调和灵活性                           |
| 指法/点穴   | 洞察、机敏   | 需要精准判断穴位和快速出手                     |
| 暗器        | 机敏、洞察   | 需要准确投掷和判断距离                         |

## 战斗系数设计指南

### 可用属性

- **战斗属性**（已乘境界）：臂力、根骨、机敏、洞察
- **固定属性**（需标记"乘境界":true）：悟性、风姿、福缘

### 设计原则

1. 系数总和必须等于品阶对应值（粗浅0.6 ~ 传说2.0）
2. 属性数量不限，可以是1个、2个、3个甚至更多
3. 高品阶功法可以加入悟性等固定属性，体现其独特性
4. 同类型功法可以有不同的属性配置，体现功法特色

### 各类型功法示例

**剑法**（核心属性：机敏、洞察）

```json
// 粗浅·普通剑法（0.6）- 单属性
{ "属性列表": [{ "属性": "机敏", "系数": 0.6 }] }

// 上乘·华山剑法（1.0）- 双属性
{ "属性列表": [
  { "属性": "机敏", "系数": 0.6 },
  { "属性": "洞察", "系数": 0.4 }
]}

// 绝世·独孤九剑（1.5）- 多属性，含固定属性
{ "属性列表": [
  { "属性": "洞察", "系数": 0.8 },
  { "属性": "机敏", "系数": 0.4 },
  { "属性": "悟性", "系数": 0.3, "乘境界": true }
]}
```

**刀法/重武器**（核心属性：臂力、根骨）

```json
// 镇派·降龙刀法（1.2）
{ "属性列表": [
  { "属性": "臂力", "系数": 0.7 },
  { "属性": "根骨", "系数": 0.5 }
]}
```

**拳掌**（核心属性：臂力、机敏）

```json
// 绝世·降龙十八掌（1.5）
{ "属性列表": [
  { "属性": "臂力", "系数": 0.8 },
  { "属性": "根骨", "系数": 0.4 },
  { "属性": "机敏", "系数": 0.3 }
]}
```

**指法/点穴**（核心属性：洞察、机敏）

```json
// 绝世·六脉神剑（1.5）- 内力外放型
{ "属性列表": [
  { "属性": "洞察", "系数": 0.6 },
  { "属性": "机敏", "系数": 0.5 },
  { "属性": "根骨", "系数": 0.4 }
]}
```

**内功**（不直接参与战斗，影响气血/内力）

- 通常不需要战斗系数，只提供资源加成
- 特殊内功（如吸星大法）可通过特性描述战斗效果

**轻功**（核心属性：机敏）

```json
// 绝世·凌波微步（1.5）- 闪避型，可加入悟性
{ "属性列表": [
  { "属性": "机敏", "系数": 0.8 },
  { "属性": "洞察", "系数": 0.4 },
  { "属性": "悟性", "系数": 0.3, "乘境界": true }
]}
```
