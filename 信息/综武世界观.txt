# 综武世界观

## 一、世界格局

### 核心设定
这是一个**多国并存**的武侠世界。金庸各部小说的门派、势力、人物**同时存在**于不同国家和地区。

### 天下大势（1200年）

| 国家 | 都城 | 统治者 | 说明 |
|------|------|--------|------|
| 大宋 | 临安 | 宋宁宗 | 偏安江南，武林正道多在此 |
| 金国 | 中都 | 完颜氏 | 占据中原，国力渐衰 |
| 蒙古 | 草原 | 铁木真 | 统一草原，虎视天下 |
| 大理 | 大理城 | 段智兴 | 段氏皇族，武学世家 |
| 西夏 | 兴庆府 | 李氏 | 西北边陲，一品堂高手众多 |
| 吐蕃 | 逻些 | 各部 | 密宗武学，金轮法王崛起 |
| 大辽（残部） | 西辽 | 耶律氏 | 契丹遗民，萧峰传说流传 |

## 二、各势力分布

### 正道门派

**少林寺**（嵩山）：
- 方丈：玄慈大师
- 天下武学正宗，七十二绝技、易筋经镇寺
- 扫地僧为寺中隐修高僧，陆地神仙境界

**武当派**（武当山）：
- 掌门：张三丰（宗师境界，即将突破绝顶）
- 太极拳、太极剑初创，武当七侠已入门
- 与少林并称泰山北斗

**全真教**（终南山）：
- 掌教：马钰（王重阳已逝）
- 全真七子各有所长
- 先天功、全真剑法传世

**丐帮**（总舵流动）：
- 帮主：洪七公
- 降龙十八掌、打狗棒法为镇帮绝学
- 天下第一大帮，弟子遍布

**五岳剑派**：
- 嵩山派：左冷禅（野心勃勃，图谋并派）
- 华山派：岳不群（气宗剑宗之争暗流涌动）
- 衡山派：莫大先生
- 恒山派：定闲师太
- 泰山派：天门道长

**峨嵋派**（峨嵋山）：
- 掌门：风陵师太（郭襄尚未创派，此为前代传承）
- 峨嵋九阳功、峨嵋剑法

**大理段氏**（大理）：
- 皇帝：段智兴（南帝，绝顶境界）
- 一阳指传世，六脉神剑仅段誉一人会使
- 段誉已年过百岁，隐居天龙寺，陆地神仙境界

**逍遥派/灵鹫宫**（天山）：
- 宫主：虚竹（陆地神仙境界，年过百岁）
- 北冥神功、天山六阳掌、生死符
- 隐世不出，偶收有缘人

### 邪道势力

**明教**（光明顶）：
- 教主：阳顶天
- 四大法王：紫衫龙王、白眉鹰王、金毛狮王、青翼蝠王
- 乾坤大挪移、圣火令武功
- 与中原武林对立，实则亦正亦邪

**日月神教**（黑木崖）：
- 教主：任我行
- 副教主：东方不败（尚未练葵花宝典）
- 吸星大法、葵花宝典（藏而未练）
- 与五岳剑派争斗不休

**星宿派**（星宿海）：
- 掌门：丁春秋（宗师境界）
- 化功大法、神木王鼎
- 弟子谄媚成风

**神龙教**（神龙岛）：
- 教主：洪安通
- 势力尚小，盘踞海外
- 豹胎易筋丸控制教众

**铁掌帮**（铁掌峰）：
- 帮主：裘千仞（宗师境界）
- 铁掌水上漂名震江湖

**古墓派**（终南山活死人墓）：
- 传人：小龙女（尚年幼，由师父抚养）
- 玉女心经、玉蜂针
- 与全真教比邻而居，互不往来

### 其他势力

**天地会**（秘密组织）：
- 总舵主：陈近南
- 以反金复宋为旗号
- 红花会尚未成立

**桃花岛**（东海）：
- 岛主：黄药师（绝顶境界）
- 弹指神通、落英神剑掌、碧海潮生曲
- 黄蓉为唯一传人

**白驼山**（西域）：
- 山主：欧阳锋（绝顶境界）
- 蛤蟆功、灵蛇拳
- 图谋九阴真经

**姑苏慕容**（燕子坞）：
- 家主：慕容复（已疯癫多年，由家仆照料）
- 斗转星移、参合指
- 家道中落，仅余旧仆

## 五、武学传承

### 少林一脉
```
达摩祖师
└→ 少林寺（七十二绝技、易筋经）
   ├→ 武当派（张三丰悟太极）
   ├→ 峨嵋派（部分传承）
   └→ 各派内功根基
```

### 逍遥一脉
```
逍遥子（陆地神仙，已隐世）
├→ 天山童姥（天山折梅手、生死符）→ 已逝
├→ 无崖子（北冥神功、小无相功）→ 传功虚竹后坐化
├→ 李秋水（白虹掌力）→ 已逝
└→ 丁春秋（化功大法）→ 星宿派（邪道）

虚竹（灵鹫宫）
└→ 隐世天山，偶收有缘人
```

### 大理段氏
```
段思平
└→ 一阳指、六脉神剑
   ├→ 段誉（六脉神剑、北冥神功）→ 天龙寺隐修
   └→ 段智兴（一阳指）→ 南帝
```

### 丐帮传承
```
降龙十八掌
└→ 萧峰（完善）→ 历代帮主 → 洪七公

### 全真/古墓
```
王重阳（先天功、全真剑法）
├→ 全真七子

### 桃花岛

├→ 落英神剑掌

### 明教

└→ 中土明教
   ├→ 乾坤大挪移（教主专属）
   ├→ 圣火令武功

## 六、重大事件时间线

### 百年前（天龙时代）
- 萧峰雁门关自尽
- 段誉继位大理皇帝，后出家
- 虚竹继承灵鹫宫，隐居天山
- 慕容复疯癫

### 数十年前
- 王重阳创全真教
- 第一次华山论剑，王重阳夺魁
- 王重阳逝世
- 张三丰创武当派

### 当前（1200年）
- 郭靖在蒙古长大
- 铁木真统一草原
- 各派势力并存

### 未来可能
- 郭靖南下
- 第二次华山论剑
- 蒙古南侵
- 襄阳保卫战
- 各势力冲突与合作

## 七、特殊设定

### 高手寿命
- 境界越高，寿命越长
- 宗师可活百岁
- 绝顶可活百二十岁
- 陆地神仙寿命不可测，虚竹、段誉已过百岁仍健在

### 天龙遗老
- 虚竹、段誉作为天龙时代的幸存者，已是传说级人物
- 极少出手，但其存在本身就是一种威慑
- 偶有后辈得其指点，便可一飞冲天

### 武学交融
- 各派武学在这个世界可能产生交融
- 如吸星大法与北冥神功的渊源
- 如太极与逍遥派武学的相似之处
- 玩家可能获得跨派武学的机缘
