# 新建内容规范

## 一、新建人物

### 初始属性（7维，范围0-20）
臂力、根骨、机敏、洞察、悟性、风姿、福缘

**分档参考**：
- 1-4：愚钝（负面特质）
- 5-8：普通
- 9-12：人才
- 12-16：天才（正面特质）
- 17-20：妖孽（极端正面特质）

### 境界
**大境界**：不入流 → 三流 → 二流 → 一流 → 宗师 → 绝顶 → 陆地神仙
**小境界**：初期 / 中期 / 后期 / 圆满

格式示例：`"境界": "一流-后期"`

---

## 二、新建功法

### 品阶（低→高）
粗浅 → 传家 → 上乘 → 镇派 → 绝世 → 传说

### 类型
- 内功、外功、轻功
- 武技：剑法 / 刀法 / 拳掌 / 指法 / 暗器 / 枪戟 / 棍锤

### 掌握程度（低→高）
初窥门径 → 略有小成 → 融会贯通 → 炉火纯青 → 出神入化

---

## 三、新建物品

### 类型
秘籍、装备、丹药、杂物

### 品质（低→高）
凡品 → 精品 → 珍品 → 极品 → 绝品 → 神品
