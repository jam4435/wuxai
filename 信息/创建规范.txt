# 新建内容规范

## 一、新建人物

### 初始属性（7维，范围0-20）

| 属性 | 说明 |
|------|------|
| 臂力 | 力量、负重、近战威力 |
| 根骨 | 体质、气血、抗击打、内力容量 |
| 机敏 | 速度、反应、闪避、出招速度 |
| 洞察 | 感知、观察、识破弱点、临敌判断 |
| 悟性 | 学习速度、功法领悟、触类旁通 |
| 风姿 | 气质、魅力、社交、NPC好感 |
| 福缘 | 运气、奇遇概率、掉落品质 |

**分档表现**：

| 数值 | 档次 | 表现描述 |
|------|------|----------|
| 1-4 | 愚钝 | 明显缺陷，常人之下。臂力1-4难以提起重物；悟性1-4学武事倍功半，常人三月所学需一年；洞察1-4对危险毫无察觉 |
| 5-8 | 普通 | 芸芸众生，无特殊表现。江湖中大多数人处于此档 |
| 9-12 | 人才 | 小有天赋，可造之材。臂力9-12可开硬弓；悟性9-12能举一反三；洞察9-12能察言观色 |
| 13-16 | 天才 | 百里挑一，名门弟子水准。臂力13-16力能扛鼎；悟性13-16如郭靖后期，虽愚钝但专注可成大器；洞察13-16如黄蓉，心思玲珑 |
| 17-20 | 妖孽 | 万中无一，主角光环。悟性17-20如杨过、令狐冲，一点即通，自创武学；洞察17-20如黄药师，算无遗策；福缘17-20如虚竹，奇遇不断 |

**角色参考**：
- 郭靖：臂力14、根骨16、机敏6、洞察8、悟性4、风姿8、福缘18
- 黄蓉：臂力6、根骨8、机敏16、洞察18、悟性17、风姿18、福缘14
- 杨过：臂力12、根骨12、机敏15、洞察14、悟性18、风姿17、福缘16
- 令狐冲：臂力10、根骨10、机敏14、洞察12、悟性17、风姿15、福缘15

### 境界表现

| 境界 | 代表人物 | 战力表现 |
|------|----------|----------|
| 不入流 | 普通人、初学武者 | 略强于常人，能胜三五个壮汉。招式生疏，内力微弱或无 |
| 三流 | 普通镖师、山贼头目 | 可以一敌十，刀枪难伤。有基础内力，招式初成 |
| 二流 | 各派普通弟子、江湖散人 | 可以一敌百，寻常兵器难伤。内力可外放，轻功可跃屋 |
| 一流 | 各派长老、江南七怪 | 江湖成名人物，可力敌精锐小队。内力深厚，招式精妙，轻功可跃数丈 |
| 宗师 | 全真七子、江南七怪合力 | 一派之尊，可力敌百人精锐。内力浑厚如海，一招一式皆有宗师气象 |
| 绝顶 | 五绝、金轮法王 | 天下有数的绝顶高手，可屠城灭国。内力无穷，武学已臻化境，一人可敌万军 |
| 陆地神仙 | 逍遥子、阿青、独孤求败 | 超凡入圣，已非凡人。阿青一人败三千甲士，独孤求败无敌于天下。此境界者，史书留名，百年难遇一人 |

**小境界说明**：
- 初期：刚突破，根基未稳
- 中期：境界稳固，开始精进
- 后期：即将触摸上一境界门槛
- 圆满：本境界大圆满，只差一步突破

## 二、新建功法

### 品阶表现

| 品阶 | 代表功法 | 说明 |
|------|----------|------|
| 粗浅 | 基础拳脚、江湖野路子 | 无门无派的粗浅功夫，街头混混所学。无特性 |
| 传家 | 越女剑法、空明拳 | 小门派传承、世家绝学。有1个特性（出神入化解锁） |
| 上乘 | 全真剑法、打狗棒法 | 名门正派核心功法，江湖上有名号。有2个特性 |
| 镇派 | 蛤蟆功、一阳指、先天功 | 一派镇山之宝，掌门级功法。有3个特性 |
| 绝世 | 降龙十八掌、九阴真经、乾坤大挪移 | 天下闻名的绝学，得之可称雄武林。有4个特性 |
| 传说 | 独孤九剑、北冥神功、易筋经 | 传说中的神功，可开宗立派、改天换命。有5个特性（每个掌握程度各一） |

### 类型
内功、外功、轻功、剑法、刀法、拳掌、指法、暗器、枪戟、棍锤

### 掌握程度表现

| 程度 | 表现描述 |
|------|----------|
| 初窥门径 | 刚学会招式，生疏僵硬，需刻意回忆才能使出。如郭靖初学降龙十八掌，有形无神 |
| 略有小成 | 招式熟练，可在实战中运用，但变化不足。如普通弟子练功十年的水平 |
| 融会贯通 | 招式已成本能，可随机应变，开始理解功法精髓。如杨过初悟玄铁剑法 |
| 炉火纯青 | 收发自如，招式之间毫无滞涩，已得功法七八分真意。如洪七公的降龙十八掌 |
| 出神入化 | 人剑合一，已臻化境，可在原有基础上推陈出新。如独孤求败的独孤九剑，无招胜有招 |

## 三、新建物品

### 类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| 秘籍 | 记载功法的书籍、图谱 | 九阴真经残卷、武穆遗书 |
| 装备 | 兵器、护甲、饰品 | 玄铁重剑、软猬甲、玉蜂针 |
| 丹药 | 可服用的药物 | 九花玉露丸、通脉丹、断肠草 |
| 杂物 | 其他物品 | 银两、信物、地图、食物 |

### 品质表现

| 品质 | 稀缺度 | 说明 |
|------|--------|------|
| 凡品 | 随处可见 | 普通铁匠铺可买，寻常药铺可得。如普通长剑、金创药 |
| 精品 | 城镇可寻 | 需在大城镇专门寻找，价值数十两。如精钢长剑、上等伤药 |
| 珍品 | 一城难求 | 需要门路或机缘，价值数百两。如名匠打造的兵器、珍稀药材 |
| 极品 | 江湖罕见 | 有价无市，需特殊渠道。如倚天剑、屠龙刀级别以下的名器 |
| 绝品 | 天下有数 | 整个江湖屈指可数，得之可改变命运。如倚天剑、屠龙刀、玄铁重剑 |
| 神品 | 传说之物 | 只存在于传说中，或已失传。如轩辕剑、干将莫邪 |

### 物品类型×品质参考

**秘籍**：
- 凡品：江湖流传的基础拳谱
- 精品：小门派入门功法
- 珍品：名门正派外传功法
- 极品：一派不外传的核心功法
- 绝品：九阴真经、九阳真经级别
- 神品：传说中已失传的上古功法

**装备**：
- 凡品：铁匠铺普通兵器
- 精品：名匠打造的精良兵器
- 珍品：有来历的名器
- 极品：江湖上有名号的神兵
- 绝品：倚天剑、屠龙刀、玄铁重剑
- 神品：传说中的神兵利器

**丹药**：
- 凡品：普通金创药、解毒丸
- 精品：上等伤药、补气丹
- 珍品：可疗重伤、增进功力的丹药
- 极品：可起死回生、大幅增进修为
- 绝品：可脱胎换骨、重塑根骨
- 神品：传说中可令人突破境界的仙丹
